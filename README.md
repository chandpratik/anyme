## Tech Stack

**Client:** React, Redux, Axios, React Router.

## Run Locally

Clone the project

```bash
  git clone https://github.com/chandpratik/anyme.git
```

Go to the project directory

```bash
  cd anyme
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
